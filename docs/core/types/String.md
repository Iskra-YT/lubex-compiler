# String

**Represents a sequence of characters stored in memory.**

## Description

The `String` type is used to store and manipulate text data.

## Syntax

```lubex
let s: String = "Hello, World!";
let name: String = "Lubex";
```

## Usage
Use String for working with text, such as:
- storing user input
- manipulating text data

## Notes
- Strings are always null-terminated by default
- String literals may be stored in read-only memory
- Accessing out-of-bounds indices results in undefined behavior
